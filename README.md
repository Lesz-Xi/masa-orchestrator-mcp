[masa-epistemic-orchestrator](https://github.com/Lesz-Xi/masa-epistemic-orchestrator)

## Status

This repository is kept for historical reference only.

It reflects an older MASA MCP design centered on:

- the original transport and operator-console setup
- earlier delegation and compliance workflow surfaces
- pre-v2 orchestration assumptions

It does **not** represent the current public epistemic-orchestration architecture.

## Use The New Repository For

- the current Claude and Codex MCP integration model
- authoritative served-reference provenance enforcement
- deterministic fallback retries
- the task-scoped runtime boundary via `TaskExecutionSession`
- the current public docs, tests, and deployment guidance
